AI Reviewer REST API 接口说明

API 概述

AI Reviewer 提供了完整的 REST API，支持代码审查的自动化集成。所有API端点都返回JSON格式的数据。

基础信息

  • Base URL: http://localhost:8080
  • Content-Type: application/json
  • 认证方式: Token 或 OAuth2

响应格式

{
  "success": true,
  "data": { ... },
  "message": "操作成功",
  "timestamp": "2024-01-01T00:00:00Z"
}

API 端点

GET /api/health

参数: 参数列表

使用示例

cURL 示例

# 触发代码审查
curl -X POST http://localhost:8080/api/review/trigger \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "repositoryUrl": "https://github.com/user/repo",
    "pullRequestId": "123"
  }'

# 获取健康状态
curl -X GET http://localhost:8080/api/health

JavaScript 示例

// 触发代码审查
const response = await fetch('/api/review/trigger', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer YOUR_TOKEN'
  },
  body: JSON.stringify({
    repositoryUrl: 'https://github.com/user/repo',
    pullRequestId: '123'
  })
});

const result = await response.json();
console.log(result);
